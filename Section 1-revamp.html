Lesson 1: First Code

Basics of the web.

The web consists of a collection of personl computers and servers communticating
with each other via HTTP requets. The serve "serves up" the requested HTML where
it is interprated by your browser.

HTML

Heart of the web.

HTML consists of text content, markups, references to otther documents
and links to other pages.

Computers, not so smart

Computers only have the ability to take everything
given to them literally. Mistakes matters and even little ones can cause failure.

Span and Div

Div is a tag used to create blocks of code. Span is for inline code. Span can be used within a paragraph tag,
but div should not be. Div allows for changes within a block and span for changes within a specific line

Lesson 2: The Structure Within

Block elements

Html elements are either block or inline. Block elements form a box around part of the web page your looking at.
It can be easy to visually take a web page and form boxes around each of the things you see.

Looking at the code behind the block

Html exist in a tree like structure. Elements (or complete tags) existing inside of each other. HTML uses indentions to visually represent the nesting of boxes with in boxes.

DOM

DOM or Document Object Model is the tree like structure of a webpage. A standard convention for interatcing and representing elements in HTML
Broswers turn HTML tags into elements in the tree or DOM
